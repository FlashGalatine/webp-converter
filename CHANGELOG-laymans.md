# What's New in 2.7.0

*Released: March 2, 2026*

---

## ✨ New Features

- **Multi-Crop Zone Export:** You can now select and extract multiple different regions from a single image at the same time.
- **Independent Zone Settings:** Apply different settings (like WebP quality, formats, and max dimensions) to each individual crop area.
- **Export All:** Download all of your cropped areas at once in a convenient ZIP file with a single click.

## 🔧 Improvements

- **Simplified Controls:** We have cleaned up the interface by removing duplicate export options, making the tool easier to use.
- **Streamlined Processing:** Under-the-hood changes to our image processing pipeline ensure faster and more reliable exports.
